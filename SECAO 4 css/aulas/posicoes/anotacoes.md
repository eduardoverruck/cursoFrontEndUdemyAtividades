**posicoes no html e css**

>> valor padrao é static

>> relative
top, left, right e bottom movem o elemento pela tela
>pode sobrepor os outros elementos
>ocupa o espaço que deveria mesmo se não mostrar o conteudo no seu lugar


>>absolute
está acima dos outros elementos da tela e não ocupa um espaço na mesma
